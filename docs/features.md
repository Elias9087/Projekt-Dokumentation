# Hauptfunktionen der Dokumentation

In diesem Projekt nutze ich Markdown und GitHub Pages, um Inhalte strukturiert zu präsentieren. Die wichtigsten Funktionen sind:

* **Strukturierte Inhalte:** Ich verwende unterschiedliche Überschriften-Ebenen (`#`, `##`), Listen und Code-Blöcke für eine klare Lesbarkeit.
* **Interne Verknüpfung:** Ich habe die Seiten so vernetzt, dass ich über relative Links (z. B. von der `index.md` in den `docs`-Ordner) einfach navigieren kann.
* **Automatisches Web-Hosting:** Durch die Aktivierung von GitHub Pages wird jede Änderung, die ich speichere, sofort online unter meiner Projekt-URL aktualisiert.
* **Versionsverlauf:** Da ich GitHub nutze, kann ich jede Anpassung an der Dokumentation genau nachverfolgen und bei Bedarf auf ältere Versionen zurückgreifen.
